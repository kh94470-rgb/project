정규표현식

정규표현식 객체 생성

$&, $`, $'

()소괄호와 |

플래그 문자

g : 전역 비교 수행
i : 대소문자를 가리지 않고 비교

앵커 문자

문자열의 앞^과 문자열의 뒤$를 구분해주는 정규 표현식 기호

수량 문자

a* : a가 0개 이상
a+ : a가 1개 이상
a? : a가 0개 또는 1개

{}를 이용한 수량 표현

a{5} : a가 5개
a{2,5} : a가 2~5개
a{2,} : a가 2개 이상

메타문자와 []대괄호

각각의 메타문자는 특별한 의미의 문자열 양식을 나타내고 괄호 묶음 안의 내용은 문자의 범위를 나타냄
대괄호 안의^은 시작이 아니라 특정 문자열 제외를 뜻함

추가 메타 문자

\d : 숫자
\w : 영어,숫자, _(언더스코어)
\s : 공백문자(띄어쓰기, 탭, 줄바꿈)
\D : 숫자만 빼고 다
\W : 영어, 숫자, _빼고 다
\S : 공백문자 빼고 다

Escaping

포인트 .
